# Certified in Cybersecurity (CC) Training

This repository documents my training for the **Certified in Cybersecurity (CC)** certification by ISC2, covering all key domains of the certification.

---

## Domains Covered

1. **Security Principles**  
Fundamental security concepts including CIA triad, risk management, governance, and ethics.

2. **Incident Response & Business Continuity**  
Incident handling, response lifecycle, business continuity, and disaster recovery basics.

3. **Access Control**  
Authentication, authorization, identity and access management, and access control models.

4. **Network Security**  
Network architecture, common attacks, firewalls, and secure communication protocols.

5. **Security Operations**  
Monitoring, vulnerability management, patching, and operational security best practices.

---

## Repository Structure

Each domain includes notes, labs, and resources to support learning.

---

## Summary

This training has built my foundational cybersecurity skills, preparing me for entry-level roles in security operations, cloud security, and DevOps.
